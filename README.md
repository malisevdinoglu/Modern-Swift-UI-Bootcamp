## ğŸš€ Setup & Installation

### Prerequisites
- **Xcode 14+** installed
- **macOS 12.0+**
- **iOS 15.0+** deployment target
- Basic knowledge of Swift and SwiftUI

### How to Run Projects

1. **Clone the repository**
```bash
git clone https://github.com/malisevdinoglu/Modern-Swift-UI-Bootcamp.git
cd Modern-Swift-UI-Bootcamp
```

2. **Open a project in Xcode**
```bash
# Navigate to a project folder
cd APIExplorer

# Open the Xcode project
open APIExplorer.xcodeproj
```

3. **Select a simulator or device**
   - Choose iPhone 14 or newer from the device menu
   - Or connect a physical iOS device

4. **Build and run**
   - Press `Cmd + R` or click the Run button
   - Wait for the build to complete
   - App launches in simulator or device

### Running Swift Fundamentals
```bash
# For Swift fundamentals (Odev files)
# Open in Xcode or run in Terminal
swift Odev1.1_main.swift
swift Odev1.2_main.swift
```
